## 获取环境变量
- 本项目需要配置6个必需的环境变量
- 下方提供了详细的配置截图教程，如果找不到截图里的信息，可使用Ctrl+F快速定位关键信息
- 遇到任何问题欢迎在 [GitHub Issues](https://github.com/scoful/pocker/issues/new) 中反馈
- 环境变量配置示例：
```
ACCESS_KEY=xxxxxx # 系统登陆密钥
GITHUB_TOKEN=xxxxxx # github 的 token
HUAWEICLOUD_AK=xxxxxx # 华为云 ak
HUAWEICLOUD_SK=xxxxxx # 华为云 sk
NEXT_PUBLIC_HUAWEICLOUD_USERNAME=xxxxxx # 华为云容器镜像服务 SWR 登陆用户名
HUAWEICLOUD_PASSWORD=xxxxxx # 华为云容器镜像服务 SWR 登陆密码

```
### ACCESS_KEY
- 系统登录凭证，请设置一个安全的密钥值，用于系统身份验证和访问控制(也就是登陆密码)
### GITHUB_TOKEN
- 访问[GitHub令牌设置](https://github.com/settings/tokens/new)创建一个新的访问令牌，有效期建议设置为永久，**_注意：权限仅需勾选 workflow 即可_**

### HUAWEICLOUD_AK
- 访问[华为云控制台](https://console.huaweicloud.com/iam/)获取ak，创建一个新的“用户组”并只授予**SWR FullAccess**权限，然后在这个用户组下创建一个新的“用户”，**_注意：创建时只需授予 SWR FullAccess 权限即可_**
- **_注：ak 和 sk 创建后，立刻找地方保存，如果错过保存，重新创建获取。_**

## 修改 Vercel 环境变量
- 打开 [Vercel](https://vercel.com/)
- 在 Vercel 项目控制面板中，找到一键部署后的项目，进入 Settings → Environment Variables 页面
- 找到需要修改的环境变量，直接修改
- 修改完成后，点击 Redeploy 重新部署项目
- 部署完成后即可开始使用系统，尽情体验吧！
